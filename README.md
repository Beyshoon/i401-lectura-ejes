# Actividad Lectura de eje X - eje Y

## Objetivo

Crear una interfaz web utilizando Flask para mostrar datos simulados de los ejes X e Y.

## Observaciones

Actividad en clases, junio 2026.

Desarrollado en AmogOS utilizando Python, Flask, HTML y CSS.

## Funcionamiento

La aplicación genera valores aleatorios para eje X y eje Y, los muestra en una página web personalizada y actualiza los datos automáticamente cada 2 segundos.
